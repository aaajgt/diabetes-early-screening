# 糖尿病早期筛查机器学习模型

## 项目简介
本项目基于机器学习算法构建糖尿病早期筛查模型，对比了逻辑回归、随机森林等经典模型的性能，旨在为临床糖尿病早期筛查提供高精度、易部署的工具。项目包含完整的数据预处理流程、模型训练/调参/评估代码，以及可视化分析结果，可直接复现所有实验结论。

### 核心功能
- 数据清洗与特征工程（处理缺失值、异常值、特征编码等）；
- 逻辑回归/随机森林模型的训练、超参数调优；
- 模型性能评估（混淆矩阵、ROC-AUC、PR曲线、特征重要性分析）；
- 不同模型的性能对比与场景适配性分析。

## 数据说明
### 数据来源
- 原始数据：采用[Kaggle Pima Indians Diabetes Dataset](https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database)（可替换为实际使用的数据集链接）；
- 数据说明：包含768条样本，8个特征（如血糖、胰岛素、BMI、年龄等），目标变量为是否患糖尿病（0=未患病，1=患病）。

### 数据文件
- `data/raw_data.csv`：原始未处理数据；
- `data/processed_data.csv`：经清洗、标准化、特征筛选后的处理数据；
- `data/data_source.txt`：数据来源详细说明（含下载链接）。
